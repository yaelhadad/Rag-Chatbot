# RAG Chatbot Frontend

React frontend for the RAG Chatbot system.

## Installation

```bash
cd chatbot/frontend
npm install
```

## Development

```bash
npm run dev
```

The app will start on `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- 3 RAG methods selection
- Real-time chat interface
- Source display with collapsible details
- Method-specific source rendering (graph, parent-child, entropy analysis)
- Beautiful Tailwind CSS styling
